Wiki 系统

需求：git+md

系统 实现方式 缺点 优点
dokuwiki php+文本 编辑器不好用；数据量大搜索可能有性能问题（txt） 插件丰富，txt移植方便
mediawiki php+mysql 配置复杂；visualeditor 加载慢 功能全面，visualeditor
simiki git+md 全量生成html；图片支持不好 可定制
gitbook git+md 全量生成 html ( 渲染巨慢 ) 有编辑器
